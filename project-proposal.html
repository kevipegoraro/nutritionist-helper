Nutritionist App

Project Proposal

A simple browser-based website to assist nutritionists during appointments by recording, organizing, calculating, and exporting client information.

Main Purpose

The project supports a nutritionist during client appointments. The tool keeps the consultation data structured and reduces manual work when collecting anamnesis, anthropometric, and metabolic information.

User Need

A friend who works with nutrition needs a practical way to record and manage client data efficiently during consultations.

Expected Result

The final website should work locally in the browser, be easy to navigate, and export the entered information as a text file or PDF.
